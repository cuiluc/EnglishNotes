# 一、哲学三问
1. What is the essence of language? 语言的本质是什么
    - 声音
2. What is the essence of speaking a language? 说话的本质是什么
    - 用声音造句
3. How do we gain fluency in the English language? 如何练出流利英语
    - 建立英语思维

- 人类语言共通点
    - 张口闭口说话
    - 所有发音都是 元音Vowel 和 辅音Consonant 组成

# 二、对比中英发音
## 2.1 汉语音节表 413个音节
1. 从未单独对待辅音，中文会将辅音加上元音念出来，如 p - po
2. 发音最小单位是“音节”，辅音开头，元音结尾 只能有 413 排列组合，颗粒感
3. 习惯鼻音或元音结尾
## 2.2 英语发音的特点
How many syllables does English have? / 英语有多少音节
- 但凡张嘴过一次即有一个元音叫一个音节，没有一个音节可以单独用辅音表达
- 从发音角度看，比如watt和what是同一个音
- 举例：
    理论公式：CVC(Consonant-Vowel-Consonant)即“辅音+原因+辅音”构成的单词，理论上会产生的音节数量 CVC = 24 * 20 * 24 = 11,520； VCCC、CCV 等各种组合；所以需要练好基础发音
    - 现实例外：prompts[CCVCCCC]/scripts[CCCVCCC]]/sprints[CCCVCCC]/instincts[CV]+[CCVCCCC]...
    - 英文所有排列组合，最终答案 15,831 （汉语的近40倍）
## 2.3 区别
- 所以我们的**难点**在于，不习惯一个音节以辅音结尾，所以经常会在辅音结尾的音节后加一个不存在的原因，比如 t - te，d - de
- matt -- 马特，会把辅音变成元音结尾
- 英文可以辅音+辅音 Hanks --- k + s
- 英文不强调颗粒感
- 好像越冷的地区的语言，辅音结尾越多，闭口越多，越压抑，防止热量散失，比如德语； 奔放的语言的，都是开口音结尾

# 三、国际音标 / International Phonetic Alphabet(IPA)
## 3.1 元音 / Vowels 20个 张口
- 元音才有美音，英音有区别
- 可顺畅发出
- 发音特点
    1. 可以念很大声
    2. 可以拖很长
### 单元音 12个
- 发音时嘴型固定
    长元音（5个）
    [i:] [ɜ:] [ɑ:] [ɔ:] [u:]
    短元音（7个）
    [ɪ] [e] [æ] [ʌ] [ɒ] [ʊ] [ə]
### 双元音 8个
- 发音会在两个元音间自然滑动
    [eɪ] [aɪ] [ɔɪ] [əʊ] [aʊ] [ɪə] [eə] [ʊə]
- [ə] 重音变成 [ʌ]

## 3.1' 元音发音练习
### 3.1‘.1 [ə] 所有元音的原点
- [ə] 呃，所有元音的嘴形起点
    - 最简单的一个音：微张着嘴就行！**不是重音的音节中的元音很多都可弱化成[ə]**
    ```
    about，agenda，woman，women
    I can do this, Katrina, but 美国[ə] 英国[ʌ]情感更强
    ```
- [ʌ]
    - [ʌ]和[ə]一样，舌头完全放松，区别在于发音时下巴稍微往下拉一点，但嘴形不需要很大，稍微比[ə]用点力
    ```
    but

    bus,cup,cut,gun,fun,luck,lunch,run,summer,
    structure,Sunday,thunder
    ```
- [ɜ:] 英国[ə] 美国,每个都发音[ər]
    ```
    Burt/Albert

    were,perfect,serviice,verb,
    early,earth,heard,search,
    hurt,surface,Thursday,urban
    ```
### 3.1‘.2 肌肉张弛有度
1. [ɪ]松 vs [i:]紧
    - 感觉肌肉的 松[ɪ] 和 紧[i:]
    - [i:] 就是长‘衣’，绷紧嘴把拱起的舌头前部往前往上顶
    - [ɪ] 嘴巴放松（略微比[i:]嘴形大一点），试着在这种状态发[i:]，发音接近‘诶’，面部肌肉放松
    ```
    shit    sheet
    bitch   beach
    slip    sleep
    hit     heat
    sick    seek
    pitch   peach

    Lily
    ```
2. [ʊ]松 vs [u:]紧
    - [ʊ] 学[ə]面瘫去念，不要用力，嘴形比[ɪ]大一点
    - [u:] 用力
    - [ʊ] `put took could would should book look cook good foot` 
   ```
   fool     full
   pool     pull
   shoe     should
   loop     look
   ```
4. [æ]紧 vs [e]松
    - [æ] 啊 的时候变成 哎，英文中罪行最大的元音，声音相对最大，所以表现的情感最激烈，如absolutely, spectacular
    - [e] 哎 比[i]和[ʊ]一样都是非常放松的状态下发出的音，嘴只比[ə]音稍微张大一点
    - [æ] `sad bad absolute spectacular nasty`
    - [e] `red leg pen vest nest`
    ```
    bad     bed
    sad     said
    flash   flesh
    gas     guess
    tack    tech
    ```
## 3.2 辅音 / consonants 28个 闭口,
> 因为加括号的那四个有的不认可，所以 也说24
- 辅音，美音，英音没有区别
- 只能轻声念，会大声，是因为中国人会加一个[ə]，只有闭嘴爆破的一瞬间是辅音
- 念起来很短促
- 发音比较轻松，但是在实际发音过程中需要控制，反而比较费力
- 前半程 + 后半程，前半程嘴型后半程发音，只嘴型也可，所以这时变得省力
### 清辅音 / unvoiced consonants 11个
- 发音时声带不振动
    [p] [t] [k] [f] [θ]
    [s] [ʃ] [h] [tʃ] ([ts] [tr])
### 浊辅音 / voiced consonants 17个
- 发音时声带振动，手指在喉咙处感觉明显
    [b] [d] [g] [v] [ð] [z] [ʒ] [r] [j]
    [w] [m] [n] [ŋ] [l] [dʒ] ([dz] [dr])

## 3.2' 辅音发音练习
- 中文习惯鼻音或元音结尾，而英文大量的单词是辅音结尾，所以需要对辅音结尾的发音进行刻意练习
### 3.2'.1 辅音结尾“不漏气”：收放自如 [t] [d] [k] [g] [p] [b]
- **不漏气**
    - 体会一下嘴巴先憋一点气，然后再释放的感觉
    - 辅音一定要发的干净利索，不要拖泥带水
- 发音练习：收放自如，在做到“**不漏气**”的前提下：
    - 放：单词结尾的辅音清晰发出来
    - 收：结尾的辅音不放出来，但是嘴型做到位，**不是直接不要，有一个停顿，其实有一点点轻轻的音（有点儿意思）**
- 注意**不要发出元音**
- **只有这六个词汇收音**
> **练习**
1. **总** 第一组 [t] [d] [k] [g] [p] [b]
    - **辅音放，前后都做**
    - **辅音收，只做前半程**
    - **只有这六个词汇收音**
    ```
    wha`t` 
    di`d`
    loo`k`
    bi`g`
    kee`p`
    jo`b`
    ```
2. 第二组 [t] [d] 
    - 前半程都是舌尖抵住上牙齿，还有[n]抵住发音
    - [t] 清辅音
    - [d] 浊辅音，更费力
    - 不是直接跳过
    - 舌头咬到位
    ```
    just great
    most different
    I can't write that
    a night without hot chocolate

    good food / great food
    did need
    I would do that
    I'd like to take that
    ```
3. 第三组 [k] [g] 前半程一致，一清辅音，一浊辅音
    ```
    look back
    walk back
    make music
    make black music

    hug
    rag dog
    big bag
    zig zag
    ```
4. 第四组 [p] [b] 前半程一致，一清辅音，一浊辅音
    ```
    deep drop
    help chop
    cheap shop
    flip flop

    Hey Bob!
    describe
    lab tube
    ho`b`no`b` **一个词里面收两次**
    ```

- 气流摩擦 感觉气流由唇齿间的缝隙中通过 (加粗部分为比较难的辅音)

    - **[v]** 
    - **[s]**
    - **[z]**
    - **[θ]**
    - **[ð]**
    - **[ʃ]** 
    - **[ʒ]**
    - [h]

    - [tʃ] 吃 的 辅音ch
    - [dz] 滋 的 辅音z；[d] [z] 快速连读
    - [ts] 次 的 辅音c；[d] [s]] 快速连读

#### 发音练习 最难辅音（加粗为重点练习对象，不过是中文没有对应发音）
- **[θ]** **[ð]** three, those
- **[ʃ]** **[ʒ]** ship, vision
- [l] lip, small
- [m] [n] **[ŋ]** mom, many, morning
- **[r]** [w] [j] rat, wit, you
> 练习
- **[θ]** **[ð]**
    - [θ]有点像 丝
    - [ð]有点像 z
    - **气流摩擦比较短,几乎可以忽略不计，弹舌音/闪收音(类似翘舌音)**
    - 弹出的舌头/抵住牙齿 瞬间收回来，收回来一瞬间发音，气流从上齿和舌尖的缝隙摩擦成音
    - 只有在末尾的时候，多停留一会儿收回
    - 前半程一致，一清辅音[θ]，一浊辅音[ð]
    - 舌头放到牙齿中间，舌尖略微从两排牙齿中间伸出
    ```
    third, thirty, thirteen
    thousand, mouth, thank
    thirteen thousand and thirty three

    those[ðəʊz], these[ði:z], their, 
    theirs, them, thought
    bro`th`er from ano`th`er mo`th`er
    ```
- **[ʃ]** **[ʒ]** 
    - [ʃ] 清辅音，“谁”的辅音部分
        ```

        ```
    - [ʒ] 浊辅音，有点像 “日，更像 yeah 的 yea”

- [l]
    - 类似“啦”，舌头抵在牙齿上**还未下移时**发出的辅音
    - la 的 l 拖长，舌头顶一下，稍微发一下

- [m] [n] **[ŋ]** 
    - [m] ma 双唇抿住
    - [n] na 不要闭拢嘴唇，但舌尖要抵住上牙,不要发出a。 和 [t] 发音一样，然后发出 “嗯” 
    - **[ŋ]** 舌后部抬起，气流不从口腔泄出，让气流**从鼻腔**泄出，震动声带。嘴巴张大版的[n]。

- **[r]** [w] [j]
    - [r] 入，去掉元音，只做 嘴形
        - 英国 开头发音，末尾不发音
        - 美国 争取每个字母有发音
        `near father never under paper together car river four here`
        - 位于元音前时，嘴巴向前撅，舌头向后缩，舌尖向上卷
        - 位于元音后时，嘴巴不用向前撅
    - [j] 要，y
        `you your year young yes yet yard yourself yellow`
        - 双唇微开，舌尖轻轻抵住下齿龈，舌头中间的部位翘起，振动声带

# 注意
- 开始时别扭，可以把稍微把舌头伸出来多一点，练习多了之后就到自然嘴型
- 在保证发音准确度的前提下，慢慢加快速度练习，直至自然流畅发出
- 多练习熟悉声音，自己发音标准，也能精准听到老外
